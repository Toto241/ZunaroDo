# Closed Test einrichten (12 Tester / 14 Tage)

Kurzanleitung — Details in SecBrain `zunarodo-closed-test-playbook`.

## Automatisch pruefen

```powershell
cd D:\Tools\ZunaroDo
.\scripts\prepare-closed-test.ps1
```

## Schritte

1. Google Group: `zunarodo-closed-testers@googlegroups.com`
2. Play Console → Testing → Closed testing → AAB hochladen
3. Tester-Gruppe zuweisen, Opt-in-Link verteilen
4. 14 Tage warten (≥12 opted-in)
5. Nachweis: `release/closed-test-YYYY-MM.md` (Vorlage: `CLOSED_TEST_EVIDENCE_TEMPLATE.md`)

## Einladungstext (Kurz)

Betreff: ZunaroDo — geschlossener Play-Test

Bitte Opt-in-Link auf dem Android-Handy oeffnen, „Tester werden“, App aus dem Play Store installieren, 14 Tage im Test bleiben.
