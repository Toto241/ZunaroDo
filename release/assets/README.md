# Release-Belege (Play Console)

Ablage fuer Nachweis-Screenshots/-Exporte aus der Play Console, auf die
die Dokumente unter `release/` verweisen - z. B. verlangt
`release/closed-test-2026-05-30.md` hier den Konsolen-Beleg
`closed-test-2026-05-30.png` (Tester-/Zeitraum-Ansicht des Closed Tracks).

Belege erst ablegen, wenn sie echte Console-Daten zeigen; vorbereitete
Soll-Werte gehoeren nur in die Markdown-Vorlagen.
